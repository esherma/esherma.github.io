How It Works

Three steps, one afternoon

01
Extract clips from YouTube
Search, download, and interactively slice clean speech segments with automated noise reduction and loudness normalization.
02
Clone the voice
Pass a 10–15 second reference clip and its exact transcript to Qwen3-TTS. One clip is all you need.
03
Hook into Claude Code
A Claude Code Stop hook intercepts every response and pipes it through the cloned voice — no extra commands required.
